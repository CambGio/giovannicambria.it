# giovannicambria.it

Sito personale di **Giovanni Cambria — Studio di consulenza**: vetrina, blog editoriale e (in arrivo) pipeline di pubblicazione su LinkedIn.

## Stack

- [Next.js 16](https://nextjs.org) (App Router) + React 19
- TypeScript
- Tailwind CSS v4
- Hosting su [Vercel](https://vercel.com)

## Sviluppo locale

```bash
npm install
npm run dev
```

Apri http://localhost:3000.

## Script

| Script           | Cosa fa                       |
| ---------------- | ----------------------------- |
| `npm run dev`    | Dev server (Turbopack)        |
| `npm run build`  | Build di produzione           |
| `npm run start`  | Serve la build                |
| `npm run lint`   | ESLint                        |

## Deploy

Push su `main` → Vercel costruisce e pubblica automaticamente.
